# Angular Skeleton (muy básico)

Este es un esqueleto mínimo de un proyecto Angular con un componente que muestra una lista.

Pasos para arrancar (asumiendo que tienes Node.js y npm instalados):

1. (Opcional) Instala Angular CLI si quieres usar `ng serve`:

   npm install -g @angular/cli

2. Instala dependencias:

   npm install

3. Arranca la aplicación:

   npm start

Notas:
- Este esqueleto es deliberadamente muy básico. Puedes modificar los archivos en `src/app` para ampliar la app.
- Si no quieres instalar Angular CLI globalmente, puedes usar npx: `npx ng serve --open`.
